Full API¶

attenuator Module for Attenuator and related classes.
beam_stats
epics_motor Module for LCLS’s special motor records.
inout Module to centralize code related to devices that can be IN or OUT.
ipm Module for the IPM intensity position monitor class
lens Basic Beryllium Lens XFLS
lodcm LODCM: Large offset dual-crystal monochrometer.
mirror Offset Mirror Classes
movablestand Module for stands that can be moved
mps Module for devices that are integrated with the MPS system.
mv_interface Module for defining bell-and-whistles movement features
pim Profile Intensity Monitor Classes
pulsepicker Module for the LCLS1 PulsePicker
signal Module to define ophyd Signal subclass utilities.
slits Module to define Slits classes.
state Module to define positioners that move between discrete named states.
valve Standard classes for LCLS Gate Valves
